# marcobelloni

## Dove caricare le immagini slideshow

Metti le foto cantiere qui:

`content/photos/slideshow/`

Formati supportati: `.jpg`, `.jpeg`, `.png`, `.webp`

## Pubblicazione veloce

Dalla cartella progetto esegui:

`./publish.sh`

Lo script aggiorna automaticamente `content/slides.json`, fa commit e push su GitHub.
